# Data Sources

## BLS QCEW

Quarterly Census of Employment and Wages. Use open downloadable CSV files for bulk county, MSA, state, national, and NAICS data instead of repeated API calls.

## BLS LAUS

Local Area Unemployment Statistics. Used for unemployment rate, labor force, employed, and unemployed.

## Census ACS 5-year

Best Census source for small geographies such as counties. Used for population, income, education, poverty, commuting, and housing metrics.

## BEA Regional Accounts

Used for county GDP, personal income, and per-capita personal income.

## Optional future sources

FRED, building permits, job postings, and night lights can be added behind new service modules and database tables without changing the core county profile route.
